# Personal global skills

- rules must be simple, concise, precise.
- use "via negativa" style, try to start the rule with "Never". Instruct the preferred alternative at the end, if there's any.
- Never add a new skill in this repo without symlinking it into `~/.cursor/skills/`; create the symlink as part of the change.
- Never delete a skill from this repo without deleting its symlink in `~/.cursor/skills/`; remove the symlink as part of the change.
